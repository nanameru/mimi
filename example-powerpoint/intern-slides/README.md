# Bakery Pain Perdu Business Plan - HTML Recreation

このディレクトリには、「ベーカリー パンペルデュ」の焼き菓子専門ECサイト立ち上げ事業計画書をHTMLで完全再現したファイルが含まれています。

## 📁 ファイル構成

```
intern-slides/
├── bakery-business-plan.html  # メインHTMLファイル（10スライド）
├── design-reference.json       # デザイン参照データ（カラー、フォント等）
├── 名称未設定のデザイン.pptx    # オリジナルのPowerPointファイル
└── README.md                   # このファイル
```

## 🎨 デザイン仕様

### カラーパレット
- **Primary Blue**: `#4F81BD` - メインアクセントカラー
- **Dark Blue**: `#1F497D` - タイトル・ブランドカラー
- **Red Accent**: `#C0504D` - セカンダリアクセント
- **Green Accent**: `#9BBB59` - 成長・自然を表現
- **Orange Accent**: `#F79646` - 温かみの演出
- **Light Cyan**: `#4BACC6` - グラデーション用
- **Light Beige**: `#EEECE1` - 背景色

### フォント
- **タイトル・本文**: Noto Sans JP (regular, 400 / medium, 500 / bold, 700 / black, 900)
- **英語テキスト**: Calibri (regular, 400 / bold, 600)
- **フォールバック**: Arial, sans-serif

### スライドサイズ
- **寸法**: 960px × 540px (16:9 landscape)
- **レイアウト**: 縦スクロール形式で複数スライドを表示

## 📋 含まれるスライド

1. **タイトルスライド** - 「焼き菓子専門ECサイト 立ち上げ事業計画書」
2. **ブランドについて (001)** - 店舗の紹介と事業背景
3. **背景・現状の課題 (002)** - オンライン販売の必要性
4. **事業の概要 (003)** - ECサイトの特徴（3つのポイント）
5. **ターゲットと市場ニーズ (004)** - ターゲット層の分析
6. **商品ラインナップ (005)** - 3つの主力商品紹介
7. **販売と運用のしくみ (006)** - 注文から発送までのフロー
8. **予算と収益予測 (007)** - 初期費用と売上目標
9. **今後の展望と目標 (008)** - 成長戦略と将来計画
10. **ご協力のお願い (009)** - 最終スライド・連絡先情報

## 📝 注意事項

## 🛠️ カスタマイズ例

### カラーテーマの変更
HTMLファイル内のCSSセクションで、以下の変数を変更できます：
- `.gradient-bg-blue`: メイングラデーション
- `.slide-title`: タイトルカラー
- `.flow-number`: フロー番号の背景色

### コンテンツの変更
各スライドの`<div class="slide-container">`内のテキストを編集することで、内容をカスタマイズできます。
